🚀 HOW TO REPLACE IMAGES (3 EASY STEPS)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Go to image folder
        → Open: public/images/[category]/

STEP 2: Replace image file
        → Delete old image
        → Drop in your new image (keep same filename!)

STEP 3: Refresh browser
        → New image appears instantly
        → NO CODE CHANGES NEEDED!

Examples:
  Hero → Replace: public/images/hero/hero-main.png
  Facilities → Replace any file in: public/images/facilities/
  Team → Replace: public/images/team/principal.png
